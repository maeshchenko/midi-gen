## 2026-06-12 — Drift Phonk v3: по техническому заданию пользователя

- **Источник:** детальное ТЗ (атмосфера, DSP, алгоритмы партий, структура). Реализовано почти всё; сознательные отступления: tape stop в outro НЕ сделан (ломает инвариант бесшовного лупа — вместо него outro = голый рифф, шов в фильтрованное intro), bitcrusher на лиде wet 0.5 (полный делает кашу).
- **Ядро (`genres/phonk.ts`, три кастомных хука):**
  - drums: кик — шаг 0 всегда + {7,10,14}; снейр — шаг 8, 100%, halftime-закон; хэты 8-ми + пулемётные роллы (2×1/32 или триоль); build-up — accelerando снейра (половины→четверти→8-е→16-е, crescendo) с обрывом в тишину на последней доле; intro/outro без ударных.
  - bass: **строго дублирует кик** (через `ctx.shared` — новый механизм связи генераторов), легато-длительности до следующего кика → глайд 50–150мс в синте; последний кик каждого 8-тактового квадрата — октава вверх со слайдом.
  - lead: один гипнотический 2-тактовый рифф на весь трек, стаккато 16-е, onset'ы на слабых 8-х (p=0.85), словарь высот: тоника/секунда/тритон/b3/квинта/b7, октавные прыжки p=0.18, каждый чётный такт — тремоло-ролл 1/32 с crescendo.
  - Структура: intro(4)→buildup(4)→drop(8/16)→bridge(8, бас исчезает)→drop(8)→outro(2-4). Фригийский/натуральный минор, 120–135 BPM.
- **Аудио (`instruments.ts`):**
  - **Sidechain**: duck-шина (cowbell+хэты) проседает до 0.18 за 5мс на каждом кике, отпуск 130мс — реализовано через onKick-колбэк драм-кита + ramp'ы Gain.
  - **Автоматизация фильтра**: новый механизм `Ensemble.automations` (плеер вешает на Transport, рефайрится каждый круг лупа) — intro/outro: LPF лида закрыт (380Гц, «из-под воды»), buildup: линейное раскрытие до 8800Гц за секцию, drop: открыт.
  - Лид: square стаккато (decay 150мс, sustain 0) → BitCrusher(10 бит) → bandpass → LPF(автоматизируемый) → dist. Мастер: hard clip (Distortion 0.5 wet 0.7) + LPF 7500. Постоянное плёночное шипение (pink noise -42дБ).
- **Verification:** tsc чисто, vitest 83/83 — спец-тесты: снейр только на шаге 8 в дропах, бас kick-locked (±8 тиков), словарь лида, тишина в конце build-up, intro/outro = голый лид, bridge без баса. В браузере 18с прослушки через intro→buildup→drop — консоль чистая.
- **Урок:** проверки нот у границ секций — с зазором ±10 тиков (humanize-джиттер перетаскивает граничные ноты).
