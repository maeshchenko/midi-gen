### ФАКТИЧЕСКИЙ корень треска — real-time underrun буфера Web Audio (не синтез)

- **Ключевой факт от пользователя:** сохранённый **.wav чист**, хрипит ТОЛЬКО live в браузере (и игра, и UI), даже на минимуме инструментов в интро. Offline-рендер (`renderSong` через `Tone.Offline`) не имеет дедлайна реального времени → чисто. Значит проблема не в синтезе/миксе/уровнях, а в **real-time аудио-движке**: дефолтный `latencyHint: 'interactive'` берёт минимальный буфер вывода и роняет сэмплы (underrun → «хрип/треск»), когда страница параллельно рендерит игру/канвас/UI.
- **Фикс (`player.ts`):** при импорте, один раз, до захвата `Transport`, создаётся live-контекст с `latencyHint: 'playback'` + `lookAhead 0.2` (`Tone.setContext`). Больший буфер = запас аудио-потоку, underrun уходит. `Tone.Offline` строит свой контекст внутри колбэка → рендер не затронут.
- **Trade-off:** 'playback' добавляет задержку вывода (~0.2с). Для ритм-игры при необходимости снизить до числового `latencyHint` (0.05–0.1) — баланс латентность/стабильность.
- **Предыдущие аудио-правки (MetalSynth→noise, хедрум, мягче кик/компрессор) оставлены — они улучшают звук, но НЕ были причиной треска (треск был real-time underrun).**

### Фикс «цок цок» в doomerwave (тот же день)

- **Симптом (пользователь):** во всех doomerwave-треках ритмичный неприятный «цок цок».
- **Корневая причина — связка ошибок в audio-маппинге для выбранных GM-программ:**
  - lead program 90 мапился на `makeSupersawLead` (nightcore-скрим), а не на холодный синт;
  - bass program 39 → `makeBass808` (глайд-сайн + `Distortion(0.5)`) — дисторшн давал клик-транзиенты на быстрых пост-панк-восьмых;
  - kick `attack 0.001 / octaves 6` = щелчок-колотушка, в норме замаскированный, но голый в разреженном тёмном миксе doomerwave (master-lowpass 7000 режет хэты, оставляет низкий клик) → «цок» на каждый бит four-on-the-floor;
  - hatClosed был равномерным 16-м тиком (все 16 шагов 0.6/0.5) — усиливал ощущение цоканья.
- **Сделано (audio-слой, ядро/коды не тронуты по сути нот, только program-поля):**
  - `instruments.ts`: новый `softKick` в `DrumKitOpts` (pitchDecay 0.025, octaves 2.5, attack 0.006, lowpass 320 — панч без клика); doomerwave-драмы роутятся через него.
  - `instruments.ts`: новый голос `makeColdLead` (program 88) — слегка расстроенные саблы, тёмный lowpass 2400, мягкое 8-е эхо, без яркого скрима.
  - `doomerwave.ts`: lead 90→**88** (cold synth), bass 39→**38** (`makeSynthBass`, ведущий пост-панк-бас без дисторшна), hatClosed → акцентные восьмые `0.6/0.4` вместо равномерных 16-х.
- **Перепин (§2, та же сессия — ОК):** изменились program-поля и хэт-паттерн → tracks-пин `7732dcd3`→`6a8ec5cc`, снапшот genres.test переснят. vitest 144/144.

## 2026-06-12 — Вариация повторов в длинных треках (A B A B)
